# Screen Time Bridge — תוסף הדפדפן

התוסף הזה מחבר את Chrome (או Edge / Brave / Opera) לאפליקציית **Screen Time**, כך שבמקום לראות סתם "Google Chrome" בסטטיסטיקות, תראה בדיוק באיזה אתר היית — למשל **YouTube**, **GitHub**, **ChatGPT**.

הוא גם מזהה מתי וידאו או מוזיקה מתנגנים, כדי שצפייה בסרטון תיחשב כזמן מסך גם כשאתה לא נוגע בעכבר ובמקלדת.

---

## איך זה עובד (בקצרה)

1. האפליקציה Screen Time מריצה שרת זעיר מקומי על `127.0.0.1:47832` (נגיש רק מהמחשב שלך).
2. התוסף שולח לשרת הזה, בכל מעבר טאב, את כתובת האתר הפעיל והאם מתנגן בו מדיה.
3. ה-tracker באפליקציה משייך את הזמן לאתר האמיתי.

**פרטיות:** שום מידע לא יוצא מהמחשב. הכל נשאר על `127.0.0.1` המקומי.

---

## התקנה

### התיקייה להוספה

```
c:\Users\Giamat13\code\screen-time-track\chrome-extension
```

זו התיקייה שתבחר בשלב 3 למטה.

### השלבים

1. ודא ש-Screen Time **פתוחה ורצה** (גם ברקע / ב-tray זה בסדר).
2. פתח בכרום את הכתובת: `chrome://extensions`
   - ב-Edge: `edge://extensions` · ב-Brave: `brave://extensions`
3. הפעל את המתג **"Developer mode" / "מצב מפתח"** בפינה הימנית-עליונה.
4. לחץ על **"Load unpacked" / "טען לא ארוז"**.
5. בחר את התיקייה `chrome-extension` (הנתיב למעלה) ולחץ "Select Folder".
6. התוסף **Screen Time Bridge** יופיע ברשימה. זהו — סיימת.

> אין צורך להצמיד את התוסף לסרגל הכלים, אבל אפשר (אייקון הפאזל ← סיכה).

---

## בדיקה שזה עובד

1. פתח טאב ב-YouTube והתחל לנגן סרטון.
2. עבור לאפליקציית Screen Time.
3. תוך כמה שניות אמור להופיע **"YouTube"** כאפליקציה הפעילה, במקום "Google Chrome".
4. הנח לעכבר ולמקלדת בזמן שהסרטון מתנגן — הזמן אמור **להמשיך להיספר** (כי מתנגנת מדיה).

---

## פתרון תקלות

- **עדיין רואה "Google Chrome" ולא את שם האתר**
  - ודא שהאפליקציה Screen Time רצה (היא מחזיקה את השרת המקומי).
  - רענן את הטאב בכרום פעם אחת אחרי התקנת התוסף.
  - בדף `chrome://extensions` ודא שהתוסף מופעל ושאין לו שגיאות (כפתור "Errors").

- **אחרי כיבוי והדלקה של המחשב לא עובד**
  - התוסף עולה אוטומטית עם הדפדפן. רק ודא ש-Screen Time עלתה גם היא (היא מוגדרת לעלות עם Windows).

- **שיניתי קבצים בתוסף**
  - בדף `chrome://extensions` לחץ על אייקון הרענון (↻) בכרטיס של התוסף כדי לטעון מחדש.

- **רוצה לכבות את הזיהוי המפורט**
  - בהגדרות האפליקציה אפשר לכבות `browserDetail` (חזרה ל-"Google Chrome") או `countMediaWhenIdle` (להפסיק לספור מדיה בזמן idle).

---

## הערות

- עובד בכל דפדפני Chromium (Chrome / Edge / Brave / Opera / Vivaldi) — אותה תיקייה, אותו תהליך טעינה.
- Firefox לא נתמך (מנוע תוספים שונה).
- כשהדפדפן לא בחזית, התוסף מדווח "לא פעיל" וה-tracker סופר את האפליקציה האחרת שבחזית כרגיל.
